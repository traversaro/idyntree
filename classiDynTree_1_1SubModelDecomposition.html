iDynTree::SubModelDecomposition class
#include <iDynTree/SubModel.h>

Class representing the decomposition in one model in several submodels.

Contents

  • Reference
    • Constructors, destructors, conversion operators
    • Public functions

This class is used in algorithms, such as estimation of external wrenches, where a complete model is decomposed in several submodels.

For each submodel a iDynTree::Traversal is provided, so that algorithms that are explicitly designed to run on both FullModel Traversal and SubModel traversal can be executed on the SubModel.
